PuAc - Purchase Advance Council - Proiect B

  Ghid de utilizare a aplicatiei dezvoltate:

Aplicatia noastra pune la dispozitie inregistrarea/logarea a unui cont in realizarea unei achizitii(imobiliare, autoturisme, electronice, etc), iar sfatuitorii pot posta comentarii cu o nota pentru achizitia respectiva.

  • Orice achizitie poate fi identificata, de catre orice sfatuitor cu sau fara cont, printr-un cod unic.
  • Fiecare achizitie contine mai multe optiuni care pot fi filtrate.
  • Informatiile privind jurnalul tuturor activitatilor desfasurate poate fi descarcat din butonul "Descarca csv"
Tehnologiile folosite sunt: HTML5, CSS3 si Javascript. REST API cu pure nodejs si mongoDB, NOSQL DB

Structura fisierelor din client:

  • Index.html principalul fisier
  • Assets: contine toate fisierel CSS si JS
  • Images: contine imaginile folosite pentru site
  • Views: contine paginile HTML - My Items - Achizitiile mele
  • All Items - Toate Achizitiile
  • Single Item - O achizitie

Structura din server:

  • Controllers
  • Helpers
  • Models
  • Validations

Serverul este deschis la: http://localhost:5500/index.html
